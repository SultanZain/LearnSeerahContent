#### What's new in v2.0.0

* **Learn:** Introduced the new Learn experience, replacing the legacy Chapters module with a structured curriculum, bite-sized lessons, and integrated quizzes.
* **Discover:** Added Discover, an immersive feed for exploring authentic Hadith with support for sharing and exporting. Quran and Stories are coming soon.
* **Streaks:** Added a daily streak system to help build consistent learning habits.
* **Homepage:** Redesigned the homepage with a modern, responsive layout, including Daily Goal and Continue Learning cards.
* **About Us:** Refreshed the About Us page with an updated design.
* **Home Screen Widgets:** Added Android Home Screen widgets to display your learning streak.
* **User Experience:** Introduced haptic feedback on Android for a more responsive experience.
* **Offline Support:** Improved offline functionality and overall reliability.
